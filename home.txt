1. Використати API [randomuser.me](https://randomuser.me/) для отримання списку користувачів.
2. Вивести список користувачів(ім'я та номер).
3. Додати випадаючий список (dropdown), в якому можна вибрати національність.

1. Реалізувати пагінацію через кнопки "Наступна сторінка" та "Попередня сторінка".

3. Додати індикатор завантаження при зміні сторінки.
4. При фільтрації за національністю пагінація повинна зберігатися для вибраної національності.
